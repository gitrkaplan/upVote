# upVote
A Chrome extension and app for developers who want a curated list of articles, documentation and tutorials based on programming languages.

## Usage
```
$ mongod
$ npm install
$ npm run seed
$ npm run start
```

## Languages/Libraries
- HTML5
- CSS3
- Javascript(ES2015+)
- Express
- MongoDB
- Metascraper
